# Интернет-магазин климатического оборудования Aira
Современное Fullstack-приложение для продажи и заказа услуг по установке климатической техники. Проект реализован с акцентом на чистый дизайн, удобную навигацию и адаптивность.
​Технологии
​Frontend: React, HTML5, CSS3, JavaScript.
​Backend: Node.js, Express.js.
​Database: MySQL.
## Структура проекта
- `src/` - клиентская часть (страницы, компоненты, API-слой).
- `public/` - статические файлы фронтенда.
- `backend/src/` - серверные роуты и работа с БД.
- `backend/sql/` - вспомогательные SQL-файлы.
